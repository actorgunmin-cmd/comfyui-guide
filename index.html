🔀 실제 파이프라인 흐름

Load Checkpoint → CLIP Text Encode → KSampler → VAE Decode → Save Image
가장 기본적인 텍스트→이미지 흐름이에요. Load Checkpoint로 AI 모델을 불러오고, CLIP Text Encode로 프롬프트를 처리하고, KSampler가 실제로 이미지를 만들어냅니다.
Load Image (레퍼런스) → IPAdapter Plus → ControlNet → KSampler → Upscaler
레퍼런스 이미지 스타일을 유지하면서 새 이미지 생성. IPAdapter가 스타일을 학습하고, ControlNet이 구도를 고정해요. 나노바나나 지구 이미지처럼 시리즈 작업 시 필수입니다.
Load Video → ControlNet (OpenPose) → AnimateDiff → IPAdapter → Video Combine
영상을 AI 애니메이션으로 변환하는 흐름. AnimateDiff가 프레임 간 움직임을 처리하고, IPAdapter가 스타일을 일관되게 유지해요. 결과물을 영상 파일로 저장합니다.
Generate Image → Impact Pack (Face) → Rembg (누끼) → ESRGAN Upscale → Save
인물 이미지 품질 마무리 흐름. Impact Pack이 얼굴을 자동 감지해 디테일을 향상시키고, Rembg로 배경을 제거한 후 ESRGAN으로 고해상도로 키워요.
